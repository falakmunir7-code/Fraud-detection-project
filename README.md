# Fraud Detection in Government Schemes

## Description
This project implements a fraud detection system using the Isolation Forest algorithm to identify suspicious users based on abnormal claim amounts and repeated applications.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## How to Run
1. Install libraries:
pip install pandas matplotlib scikit-learn

2. Run the file:
python fraud_detection.py

## Output
- Classifies users as Legit or Fraudulent
- Shows graph visualization
- Saves results as CSV

## Working
The system uses Isolation Forest to detect anomalies (outliers) in the data and flag suspicious users.
